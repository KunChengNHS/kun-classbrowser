The NHS Digital Classifications Browser allows you to browse and search the UK Clinical Classifications. The browser includes the following features:

  • OPCS-4.9 and OPCS-4.8 classification content. The OPCS-4 Classification of Interventions and Procedures (OPCS-4) is an NHS fundamental information standard approved for publication by the Data Coordination Board (DCB) under section 250 of the Health and Social Care Act 2012 ( Ref DCB0084 ).
  • Links to the official National Clinical Coding Standards (NCCS) OPCS-4 for the NHS in England that are updated automatically when a new release of the coding standards is published
    • PDFâ€™s of the current releases of the official NCCS reference books (ICD-10 and OPCS-4) for the NHS in England can be downloaded from the Ref Books drop down
    • Links to the Coding Clinic are not included. The latest release can be found here - Coding Clinic
    • See Resources for links to the latest official coding standards used in Northern Ireland, Scotland and Wales
  • Available from any computer with an internet connection without the need to download any applications or software
  • A powerful search facility which includes; search history, search filter, wildcard search and fuzzy match, search help, clear display of where results are located in the Classification and ability to minimise and maximise search results
  • Ability to add browser bookmarks and favourites to chapters, categories and codes
  • Automatic addition of new releases of classifications when approved
  • A Resources page which contains links to classifications information and resources
